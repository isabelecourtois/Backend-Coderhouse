1. Agregar a la vista info num de procesadores presentes

2. Ejecutar FORK y CLUSTER 
Fork
node server.js --port 8080 --serverMode Fork

Cluster
node server.js --port 8080 --serverMode Cluster

3. Ejecutar usando Forever
Listar los procesos por Forever y por sistema Operativo
forever start server.js -w --port 8080 --serverMode Cluster
Se vuelve loco y abre mil ventanas

4.Ejecutar servidor modo FORK Usando PM2
Listar los procesos por PM2 y por sistema operativo
Fork
pm2 start server.js serverModeFork --watch -- --port 8081

Cluster
pm2 start server.js serverModeCluster  --watch -i max -- --port 8081
5. Hacer pruebas de finalización de procesos fork y cluster.

6. Configurar Nginx
6.1 Refirigir todas las consultas de /api/randoms al 8081.
6.2 Modificar config para /api/randoms sean redirigidas a un cluster de servidores gestionado desde Nginx repartiéndolas equitativamente entre 8082, 8083, 8084, 8085.
